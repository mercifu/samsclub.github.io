### 使用 sqlcmd 添加登录用户并授权

1. 使用管理员权限打开 cmd

2. 连接到 sql server 实例
```
sqlcmd -s SERVER_NAME\INSTACE_NAME
```

3. 添加登录用户
```SQL
CREATE LOGIN [LOGIN_NAME] FROM WINDOWS
GO

```

4. 运行存储过程对用户进行授权
```SQL
SP_ADDSRVROLEMEMBER 'LOGIN_NAME','sysadmin'
GO
```
